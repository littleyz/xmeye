XMEye is video monitoring software for IPC,NVR and DVR.

With cloud technology, easy log in by the device's serial number to show the live view monitoring video to Android phone and do operation accordingly.

  Try it now!  Get support

  • Cloud Technology

    Support remote P2P view and playback with cloud technology.

  • Local record and snapshot

    Support Local recording and play,also snapshot and pictures searching.

  • Remote views and playback

    Support real time live view and remote playback.

  • Multi application

    Available on most platforms, including Windows, Mac, Android, iOS, and WEB.

  • Two-way Audio

    Support bidirectional talk.

  • Version updating automatically

    Support app version and firmware updating automatically.

  • PTZ control

    Support device Pan/Tile/Zoom control.

  • QR scan

    Support adding device through scanning serial number by QR code.

  • User management

    Support cloud user register and modify,local and remote device add, edit and delete.
